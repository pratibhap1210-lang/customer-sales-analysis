# customer-sales-analysis
Customer Sales &amp; Profitability Analysis using SQL
## Customer Sales Analysis

Analyzed customer purchasing behavior using SQL on retail transaction data.

### Tools Used
- SQL (SQLite)
- Python (pandas)
- Google Colab

### What I Did
- Identified top customers by revenue
- Analyzed repeat customer behavior using COUNT and HAVING
- Calculated average order value per customer
- Evaluated profitability by customer and segment
- Identified customers generating negative profit

### Key Insights
- A small group of customers drives a large portion of revenue
- Some customers generate high revenue but low or negative profit
- Average order value varies significantly across customers
